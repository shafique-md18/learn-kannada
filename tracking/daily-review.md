# Daily Review

This file will stay concise. It is the 10-minute daily revision sheet.

## Review Routine

1. Read the current must-memorize words.
2. Speak each core pattern aloud.
3. Build 5 new sentences by swapping words.
4. Do 1 mini-dialogue from memory.
5. Mark anything weak separately before moving on.

## Mini-Dialogues

### Shop

You: Coffee beku.

Shopkeeper: Bill?

You: Haudu, bill kodi.

### Building

You: Lift elli?

Security: Alli.

You: Sari.

### Delivery

Delivery agent: Neevu barthira?

You: Haudu, ivaga barthini. Swalpa kaayiri.

### Auto

You: Straight hogi. Left turn madi. Illi stop madi.

### Grocery

You: Idannu eshtu?

Shopkeeper: Fifty.

You: Idannu togothini. Bag beku. UPI ideya?

### Pharmacy

You: Colgate ideya?

Shopkeeper: Ide.

You: Ondu kodi. Receipt kodi.

### Auto Meter

You: Meter alli barthira?

Driver: Elli hogbeku?

You: Office hogbeku. Extra beda.

### Cab Pickup

Driver: Pickup elli?

You: Pickup gate horage. Reach aadmele call madi.
