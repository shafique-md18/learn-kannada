## Production Practice Sample Answers

## Roleplay Sample

Office hogbeku.

## Quiz Answers

1. `hogbeku` = need to go.
2. `barbeku` = need to come.
3. Need to go to metro.
4. Cab beda, auto beku.

## Natural Bangalore Versions

- Place + hogbeku is extremely useful with auto and cab drivers.

## Common Mistakes

- `hogi` = go. `hogbeku` = need to go.
