# Lesson 023 Answers

## Guided Practice Answers

1. Yaava route?
2. Yaava room?
3. Route gotha?
4. Shortcut ideya?

## Production Practice Sample Answers

1. Yaava route?
2. Shortcut ideya?

4. Shortcut beda bidi.

## Roleplay Sample

Route gotha? Traffic jaasti andre, main road alli hogi.

## Quiz Answers

1. `yaava` = which.
2. Which gate?
3. Do you know the route?
4. Shortcut ideya?

## Natural Bangalore Versions

- Route gotha? is casual and useful.
- Beda bidi softens "leave it/no need."

## Common Mistakes

- `yaaru` = who. `yaava` = which.
