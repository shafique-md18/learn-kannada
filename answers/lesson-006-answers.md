# Lesson 006 Answers

## Guided Practice Answers

1. Neevu yaaru?
2. Avru yaaru?
3. Naan illi.
4. Avru security.

## Production Practice Sample Answers

1. Owner yaaru?
2. Naan security alla.
3. Security alli.
4. Neevu yaaru?

## Roleplay Sample

You: Avru yaaru?

Guard: Avru owner.

## Quiz Answers

1. `yaaru` = who.
2. `naan` = I.
3. `avru` is useful because it is respectful and works for he/she/they.
4. Who are you?

## Natural Bangalore Versions

- Avru yaaru? is a polite, safe question.
- Owner yaaru? is natural when asking "Who is the owner?"

## Common Mistakes

- Use `avru` for polite reference to people instead of sounding too direct.
