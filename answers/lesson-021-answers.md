# Lesson 021 Answers

## Guided Practice Answers

1. Location kalisi.
2. Pickup illi.
3. Drop office alli.
4. Cab bandide.

## Production Practice Sample Answers

1. Pickup illi.
2. Location kalisi.
3. Location tappu.
4. Drop gate alli.

## Roleplay Sample

Pickup illi. Location kalisi. Naan gate horage.

## Quiz Answers

1. `kalisi` = send.
2. `bandide` = has come/arrived.
3. Location is wrong.
4. Drop alli.

## Natural Bangalore Versions

- Pickup illi and Location tappu are natural cab-call lines.

## Common Mistakes

- `bandide` means already arrived. Use it when the vehicle/person has come.
