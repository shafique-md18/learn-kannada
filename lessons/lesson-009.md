## Main Pattern

### Kannada Sentence

Address heli.

### Hindi Pronunciation

अड्रेस हेळि.

### Meaning

Hindi: पता बताइए.

English: Tell me the address.

## Word-by-Word Breakdown

| Word | Hindi Meaning | English Meaning |
|---|---|---|
| address | पता | address |
| heli | बताइए | tell |

## Sentence Construction Logic

The thing you want someone to tell/show comes first. The action comes second.

`Address heli` = address tell.

`Room torisi` = room show.

## Reusable Pattern

`___ heli` = Tell ___

Examples:

- Address heli.
- Number heli.
- Name heli.

`___ torisi` = Show ___

Examples:

- Room torisi.
- Idu torisi.
- Lift torisi.

## Guided Practice

Build these in Kannada:

1. Tell the number.
2. Tell the name.
3. Show the room.
4. Show this.

## Production Practice

Say these in Kannada:

1. Ask security to show the lift.
2. Ask a delivery agent to tell the address.
3. Ask a shopkeeper to tell the bill amount. Use `bill heli`.
4. Say you do not know.

## Real-Life Bangalore Roleplay

You are looking for a room in an apartment.

Ask the guard:

1. Where is the room?
2. Please show the room.

Use Kannada.
