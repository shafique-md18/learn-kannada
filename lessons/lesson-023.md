## Main Pattern

### Kannada Sentence

Yaava route?

### Hindi Pronunciation

याव रूट?

### Meaning

Hindi: कौन सा रास्ता?

English: Which route?

## Word-by-Word Breakdown

| Word | Hindi Meaning | English Meaning |
|---|---|---|
| yaava | कौन सा | which |
| route | रास्ता | route |

## Sentence Construction Logic

Use `yaava` before a noun to ask "which".

`Yaava route?` = Which route?

## Reusable Pattern

`Yaava ___?` = Which ___?

Examples:

- Yaava route?
- Yaava room?
- Yaava gate?

Knowledge:

- Route gotha? = Do you know the route?
- Shortcut ideya? = Is there a shortcut?

## Guided Practice

Build these in Kannada:

1. Which route?
2. Which room?
3. Do you know the route?
4. Is there a shortcut?

## Production Practice

Say these in Kannada:

1. Ask an auto driver which route.
2. Ask if there is a shortcut.
3. Say go by main road.
4. Say no need for shortcut.

## Real-Life Bangalore Roleplay

The driver asks if shortcut is okay.

You ask: Do you know the route? If traffic is heavy, take main road.

Use Kannada as simply as possible.
