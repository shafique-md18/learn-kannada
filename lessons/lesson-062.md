## Main Pattern

### Kannada Sentence

Innondu sala heli.

### Hindi Pronunciation

इन्नोंदु सल हेळि.

### Meaning

Hindi: एक बार और बताइए.

English: Please say one more time.

## Word-by-Word Breakdown

## Sentence Construction Logic

This lesson extends the reusable pattern `Request clarification`. Keep the sentence short first, then add time/place/person words.

## Reusable Pattern

`Request clarification`

Examples:

- Innondu sala heli.
- Repeat madi.
- Clear aagi heli.
- Kelistilla.

## Guided Practice

## Production Practice

Say these in Kannada:

1. In a real Bangalore situation: Say one more time.
2. Change the place/person/item and say: Repeat.
3. Use as the next line in a short conversation: Say clearly.
4. Combine old knowledge with today: Cannot hear.

On a noisy call, ask the delivery agent to repeat clearly.
