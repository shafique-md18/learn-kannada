## Main Pattern

### Kannada Sentence

Innondu sala heli.

### Hindi Pronunciation

इन्नोंदु सल हेळि.

### Meaning

Hindi: एक बार और बताइए.

English: Please say one more time.

## Word-by-Word Breakdown

## Sentence Construction Logic

This lesson extends the reusable pattern `Request clarification`. Keep the sentence short first, then add time/place/person words.

## Reusable Pattern

`Request clarification`

Examples:

- Innondu sala heli.
- Repeat madi.
- Clear aagi heli.
- Kelistilla.
