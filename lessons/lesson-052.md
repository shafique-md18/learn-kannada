## Main Pattern

### Kannada Sentence

Nimge yaavdu ishta?

### Hindi Pronunciation

निमगे याव्दु इष्ट?

### Meaning

Hindi: आपको कौन सा पसंद है?

English: Which one do you like?

## Word-by-Word Breakdown

| Word | Hindi Meaning | English Meaning |

| ishta | पसंद | like |

## Sentence Construction Logic

This lesson extends the reusable pattern `Nimge ___ ishta?`. Keep the sentence short first, then add time/place/person words.

## Reusable Pattern

`Nimge ___ ishta?`

Examples:

- Nimge coffee ishta?
- Nimge yaavdu ishta?
- Yaavdu beku?
- Idu better.

## Guided Practice

Build these in Kannada:

1. Do you like coffee?
2. Which one do you like?
3. Which one do you want?
4. This is better.

## Production Practice

Say these in Kannada:

3. Use as the next line in a short conversation: Which one do you want?
4. Combine old knowledge with today: This is better.
