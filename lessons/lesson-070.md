## Main Pattern

### Kannada Sentence

Order wrong address ge hogide.

### Hindi Pronunciation

ऑर्डर रॉन्ग अड्रेस गे होगिदे.

### Meaning

Hindi: ऑर्डर गलत पते पर गया है.

English: Order went to wrong address.

## Word-by-Word Breakdown

| Word | Hindi Meaning | English Meaning |

| address | गलत पता | wrong address |
| ge | को/के लिए | to/for |
| hogide | गया है | has gone |

## Sentence Construction Logic

This lesson teaches delivery recovery lines. Use one short problem sentence, then one clear request.

## Reusable Pattern

`Problem sentence + request`

Examples:

- Wrong address ge hogide.
- Correct address kalisi.
- Agent call madide.
- Reschedule madi.
