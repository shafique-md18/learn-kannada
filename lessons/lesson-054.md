## Main Pattern

### Kannada Sentence

Nanage idu chennagide.

### Hindi Pronunciation

ननगे इदु चेन्नागिदे.

### Meaning

Hindi: मुझे यह अच्छा लगा.

English: I think this is good.

## Word-by-Word Breakdown

| Word | Hindi Meaning | English Meaning |
|---|---|---|
| Nanage | मुझे | to me |
| idu | यह | this |
| chennagide | अच्छा है | is good |

## Sentence Construction Logic

This lesson extends the reusable pattern `Nanage ___ anisutte`. Keep the sentence short first, then add time/place/person words.

## Reusable Pattern

`Nanage ___ anisutte`

Examples:

- Nanage idu chennagide anisutte.
- Idu correct.
- Idu wrong.
- Try madi.

## Guided Practice

Build these in Kannada:

1. I feel this is good.

3. This is wrong.
4. Try.

## Production Practice

Say these in Kannada:

1. In a real Bangalore situation: I feel this is good.
2. Change the place/person/item and say: This is correct.
3. Use as the next line in a short conversation: This is wrong.
4. Combine old knowledge with today: Try.
