## Main Pattern

### Kannada Sentence

Room clean madi.

### Hindi Pronunciation

रूम क्लीन माडि.

### Meaning

Hindi: कमरा साफ कीजिए.

English: Please clean the room.

## Word-by-Word Breakdown

| Word | Hindi Meaning | English Meaning |
|---|---|---|
| room | कमरा | room |
| clean madi | साफ कीजिए | clean/do clean |

## Sentence Construction Logic

Use the object first, then the action. English action words plus `madi` are common for household instructions.

## Reusable Pattern

`Object + action madi`

Examples:

- Room clean madi.
- Floor clean madi.
- Dustbin alli haaki.
- Idannu tegedi.
